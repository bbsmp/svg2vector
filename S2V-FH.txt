Svg2Vector FreeHep - v2.1.0-SNAPSHOT build 170420 (20-Apr-17) for Java 1.8

Converts SVG graphics into other vector formats using FreeHep libraries, with options for handling layers

usage: s2v-fh
    --all-layers                switch on all layers
 -B,--fout-islabel              use Inkscape label in output file names
 -b,--no-background             switch off background property
    --create-directories        automatically create directories for
                                output
 -d,--output-directory <DIR>    output directory, default in layer mode is
                                current directory
 -e,--print-details             print very detailed information to stdout
 -f,--input-file <FILE>         specifies the input file (path and
                                filename)
    --fout-no-basename          do not use a basename for output files (in
                                layer mode)
 -h,--help <ARG>                application and argument help
 -I,--fout-isindex              use the Inkscape layer index in output
                                file name
 -i,--fout-index                use a continuous index in output file name
 -l,--layers                    switch on layer mode, process layers,
                                create one file per layer
 -L,--layers-if-exist           switch on layer mode, if input SVG
                                document has layers, otherwise do not
                                process layers
 -n,--not-transparent           switch off transparency
    --no-errors                 switch off error messages
 -o,--output-file <FILE>        output filename, default is the basename
                                of the input file plus target extension
    --overwrite-existing        overwrite existing files on output
 -p,--print-progress            print progress information to stdout
    --print-stack-trace         sets a flag to print the stack trace of
                                exceptions
 -q,--quiet                     puts the application in quiet mode, no
                                errors, warnings, progress, or details
                                will be printed
 -r,--bgrnd-color <COLOR>       sets a background color
 -S,--simulate                  simulate application, no output
                                directories or files will be created
 -s,--text-as-shape             convert text as shapes
 -t,--target <TARGET>           specifies a conversion target
    --use-basename <BASENAME>   use the specified basename for output
                                files
 -v,--verbose                   verbose mode for application
    --version                   application version
 -w,--print-warnings            print warning messages to stdout
